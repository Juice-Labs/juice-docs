# Creating a New Pool

1. Select your organization, then navigate to the **Pools** tile.

2. Click **Create new pool**.

    ![New Pool](/img/boost/new_pool.png)

3. Name your pool, then click **Create**.

    ![Name Pool](/img/boost/name_pool.png)
