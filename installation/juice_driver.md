## Juice Driver Installation

The Juice driver is a bit more complicated than the Renderer_Win server. The Juice driver installed on any client machine that wants to utilize a remote GPU. It is installed through a simple installer which installs almost everything required.

### Requirements
- [Vulkan SDK](https://vulkan.lunarg.com/)

### Notes

Currently, the Juice driver is only tested on VMWare Virtual Machines and a small number of physical machines. HyperV is not supported at the moment.
